🛡️ SafeLink AI – Fake Link & Scam Detector
📌 Project Overview

SafeLink AI is a cybersecurity-focused web application that helps users detect phishing links, scam messages, and suspicious content.
It analyzes input text or URLs and provides a risk level, explanation, and safety tips to protect users from online threats.

🚀 Features
🔍 Detects phishing links and scam messages
⚠️ Shows risk level: Safe / Suspicious / Dangerous
🧠 Explains why something is risky
💡 Provides safety tips
🌍 Supports multiple languages:
English 🇬🇧
Tamil 🇱🇰
Sinhala 🇱🇰
📱 Simple and responsive UI
🎯 Purpose of the Project

This project is built to:

Improve cybersecurity awareness
Help users identify online scams easily
Support students and beginners with simple explanations
🛠️ Tech Stack
Frontend: Next.js / React
Styling: Tailwind CSS
AI Logic: OpenAI / API-based analysis
Deployment: Vercel
🧠 How It Works
User pastes a link or message
System analyzes content
Detects suspicious patterns (keywords, domains, behavior)
Displays:
Risk level
Explanation
Safety recommendations
🌍 Language Support

SafeLink AI supports:

English 🇬🇧
Tamil 🇱🇰
Sinhala 🇱🇰

This helps make cybersecurity awareness accessible to local students.
